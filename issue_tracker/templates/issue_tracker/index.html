Project description

This is fully functional issue tracker project, there are 3 groups of users: admin, leader and developer.
Main featuers:

  • Adding new issues.

  • Adding comments to issues by leader and developers in projects.

  • Seeing details of issues and history of changes.

  • Seeing list of all issues in projects and issues assigned to user.

  • Seeing separete lists of active and old issues.

  • -Apply for projects and join them.

  • Option of resigning from developer and leader position in project.

Leader featuers:

  • Leader can manage list of developers in project.

  • Leader can accept and deny developers applictaions to project.

  • Leader can edit reported issue.

Admin featuers:

  • Admin can accept and deny developers and leaders applictaions to project.

  • Admin can see list of issues in all project.
